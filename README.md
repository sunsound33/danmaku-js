# フロントエンド制作物
HTML、CSS、JavaScriptを使った制作
弾幕ゲーム。<br>
いわゆるシューティングゲームです。<br>

## ゲームの起動
解像度フルHD以上推奨
ディレクトリ内で `docker-compose -up -d`　dockerコンテナ起動<br>
[http://localhost:8080](http://localhost:8080) へアクセス

## ゲームの流れ
敵からの弾幕避けつつ敵に弾幕を当てて、倒していく。

## 操作方法
|コマンド|操作|
|:-:|:-:|
|Enter| ゲームスタート or 復活|
|↑|上に移動|
|↓|下に移動|
|←|左に移動|
|→|右に移動|
|Z|弾を撃つ|
|c|進める(会話)|
|v|戻る(会話)|
